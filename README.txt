Currently only tested on Windows 10 64 bit

To use install the following:
- Python <https://www.python.org/downloads/> 3.7

- Recommened add python to enviroment variables <https://github.com/BurntSushi/nfldb/wiki/Python-&-pip-Windows-installation>

- run "python get-pip.py" (get-pip.py included in zip folder) 

- `setuptools <https://pypi.python.org/pypi/setuptools>`_
  (installed automatically if you use pip).

- `Zope Interface <https://pypi.python.org/pypi/zope.interface>`_  4.4.2 or newer.
  Installing via pip will automatically download a suitable Zope Interface.

- `pywin32 <https://pypi.python.org/pypi/pywin32>`_ is required.
  Build 223 or later is highly recommended for reliable operation.\
  
- Build Tools for Visual Studio 2017 <https://visualstudio.microsoft.com/downloads/#build-tools-for-visual-studio-2017>
  No Workloads or individual components needed

- Twisted <https://pypi.org/project/Twisted/>
	If install fails use included .whl file for "pip install Twisted-18.7.0-cp37-cp37m-win_amd64.whl"
